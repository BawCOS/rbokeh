R interface to Bokeh.

### Install

```r
# from CRAN:
install.packages("rbokeh")

# from packages.tessera.io:
options(repos = c(tessera = "http://packages.tessera.io", getOption("repos")))
install.packages("rbokeh")

# from github:
devtools::install_github("bokeh/rbokeh")
```
